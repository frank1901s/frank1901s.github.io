论行为举止与专家权利

发表于 2019-11-28 更新于 2020-05-16 阅读次数： Valine：

组织中常说的几种权利：领导权利、奖励权利、惩罚权利、专家权利、参照权利。

我最渴望获得的是后两种权利，尤其是专家权利，我喜欢那种被人崇拜的感觉。

但是拥有这些权利，不止是在专业技能上有所建树，在谈吐、行为举止上也需要注重。比如穿着打扮、形神坐姿、说话方式，甚至是笑声。

我不擅长语言沟通，这已经带给我很多麻烦，我也意识到问题的严重性，我不够沉着冷静，思维不够缜密。

不以物喜不以己悲，不一定要达到这样的境界，但是，开心的时候笑声绝对不可以太过浮夸，否则给人的感觉很可能是：此人有病。

不善言辞，需要多练习，而”笑“的方式关乎修养，更为要紧。

我需要弥补这些方面的缺点，才可能获得专家权利、参照权利。
