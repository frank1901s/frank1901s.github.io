Centos7中lampp下phpmyadmin无法连接数据库及DVWA无法创建数据库问题处理

发表于 2020-03-18 更新于 2020-05-16 阅读次数： Valine：

PhpMyAdmin无法连接数据库问题解决：

报错：
phpMyAdmin #2002 Cannot log in to the MySQL server

打开配置文件

phpMyAdmin]# vim libraries/config.default.php

修改：
$cfg['Servers'][$i]['host'] = 'localhost';
为：
$cfg['Servers'][$i]['host'] = '127.0.0.1';

DVWA数据库无法创建错误修复：

修改config.inc.php文件

要将：

$_DVWA[ 'db_server' ] = 'localhost1';
修改为：
$_DVWA[ 'db_server' ] = '127.0.0.1';

并修改数据库密码
