2. 分别安装VSCode和Python

2.1 安装VSCode

基本按照提示进行安装就可以了

2.2 python3安装

注意：开始安装时注意勾选页面下方的“Add Python 3.8 to PATH”，这样就可以免去手动添加的步骤了。
然后根据提示完成安装即可。

3. 安装插件并设置VSCode

安装好VSCode和Python，设置好Python的环境变量（如果按照2.2节要求安装则已经默认设置好了）之后，只需要打开VSCode再安装一个Python插件就可以了。
