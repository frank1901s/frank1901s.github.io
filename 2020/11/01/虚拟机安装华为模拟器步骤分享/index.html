虚拟机安装华为模拟器步骤分享

发表于 2020-11-01 阅读次数： Valine：

0 前言

备考网络工程师，需要上机练习设备配置，所以安装华为模拟器。

华为模拟器V100R002C00版本，需要安装WinPcap 4.1.3、Wireshark 1.4.3、VirtualBox 5.0.16或者相近版本，由于我的电脑上VirtualBox是6.0的版本，华为模拟器不兼容此版本，所以才想到在虚拟中安装。

安装还算顺利，顺便将安装过程截图分享。

1 安装前准备

1.Win7虚拟机环境
2.安装包文件：eNSP V100R002C00B500 Setup.exe
3.将安装包文件放到Win7虚拟机桌面

2 安装步骤

2.1 双击启动安装包程序：

2.2 选择简体中文，点击“确定”：

2.3 点击“下一步”：

2.4 选择“我愿意接收此协议”，点击“下一步”：

2.5 选择安装路径，点击“下一步”：

2.6 点击“下一步”：

2.7 点击“下一步”：

2.8 勾选“安装WinPcap 4.1.3”、“安装Wireshark”、“安装VirtualBox 5.0.16”，点击“下一步”：

2.9 点击“安装”开始安装eNSP：

2.9 eNSP安装完成之后，会弹出“WInPcap 4.1.3”的安装界面，依次点击“Next”、“I Agree”、“Install”、“Finsh”完成安装（安装进度条一闪而过，所以未截到图）：



2.10 “WInPcap 4.1.3”安装完成之后，弹出“Wireshark 1.4.3”的安装引导界面，依次点击“Next”、“I Agree”、“Next”、“Next”、“Next”（选择安装路径）、“Install”、“Next”（安装完成）、“Finish”：








2.11 ”Wireshark 1.4.3“安装完之后，弹出”VirtualBox 5.0.16“安装引导界面，依次点击”Next“、“Next”（选择安装路径）、“Next”、“Yes”、“Install”、“安装”、“安装”、“安装”、“Finish”（取消启动选项）：









2.12 安装完成，点击“完成”：
