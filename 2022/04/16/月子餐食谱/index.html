月子餐食谱

发表于 2022-04-16 阅读次数： Valine：

本文整理一套月子餐食谱（适合顺产），便于查阅。

一、临产前食谱

推荐菜肴：黄芪羊肉汤
推荐容器：砂锅
烹饪时间：2小时
所需食材：羊肉200g，黄芪15g，红枣8个，红糖20g，姜片、盐适量。
烹制步骤：

  1. 将羊肉洗净，切成3cm的方块，焯水；
  2. 红枣洗净去皮；
  3. 将羊肉块、黄芪、红枣、姜片一同放入锅中，加清水，以大火煮沸；
  4. 转小火慢炖至羊肉软烂；
  5. 出锅入盐调味。

二、顺产分娩当天饮食

第一餐：花生红枣小米粥
第二餐：紫菜鸡蛋汤
第三餐：番茄菠菜鸡蛋面

三、顺产分娩产后7天饮食

3.1 第1天饮食

早餐：
红糖小米粥 1碗
鸡蛋 1个

中餐：
米饭 1小碗
清炒瓜片 1份
什菌一品煲适量。

什菌一品煲做法

  • 什菌一品煲原料
    猴头菌、草菇、平菇、香菇各2朵，白菜心1个，葱段、盐各适量。
  • 什菌一品煲烹饪步骤
    ① 香菇洗净，切去蒂部；平菇洗净，切去根部：猴头菌和草菇均洗净，切开；白菜心掰成小棵；
    ② 锅内放入葱段，加清水或素高汤大火烧开，再放入香菇、草菇、平菇、候头菌、白菜心转小火炖煮10分钟即可。

午点：
生化汤1碗

晚餐：
千层饼 适量
清炒豆芽 1份
当归鲫鱼汤 适量

晚点：
薏仁红枣百合汤 或 牛奶红枣粥1碗

3.2 第2天饮食

早餐：
牛奶红枣粥 1碗
鸡蛋 1个
苹果 1个

中餐：
米饭 1小碗
西芹炒百合 1份
番茄菠菜蛋花汤 适量

午点：
姜丝鱼汤 适量

晚餐：
红枣莲子糯米粥或 阿胶桃仁大枣羹 1碗
番茄菜花 1份

晚点：
麻油猪肝汤 适量

3.3 第3天饮食

早餐：
芝麻烧饼 1个
豆浆莴笋汤 1碗
苹果 1个

中餐：
小米饭 1碗
土豆炖牛腩 1份
珍珠三鲜汤 适量

午点：
面包 1片
猪排炖黄豆芽汤 适量
香蕉 1个

晚餐：
红薯粥 1碗
清炒苋菜 1份

晚点：
什锦面 1碗

3.4 第4天饮食

早餐：
紫米粥 1碗
鸡蛋 1个
香蕉 1条

中餐：
米饭 1碗
生菜明虾 1份
冬笋雪菜黄鱼汤 适量

午点：
香蕉百合银耳汤 适量

晚餐：
红枣桂圆粥 1碗
番茄豆腐 1份
干贝冬瓜汤 适量

晚点：
南瓜饼 1块
牛奶 1杯

3.5 第5天饮食

早餐：
胡萝卜小米粥 1碗
鸡蛋 1个
苹果 1个

中餐：
米饭1碗
肉片炒蘑菇 或 菠萝鸡片 1份
蛤喇豆腐汤 适量

午点：
银鱼苋菜汤 适量
香蕉 1根

晚餐：
桂圆芡实粥 1碗
菜心炒猪肝 1份

晚点：
木瓜牛奶露 1杯

3.6 第6天饮食

早餐：
西兰花鹤鹑蛋汤 1碗
甜橙 1个
清炒芥蓝 1份

中餐：
黑芝麻白饭 1碗
归枣牛筋花生汤 或 益母草木耳汤 适量

午点：
荔枝粥 1碗

晚餐：
鸭肉粥 1碗
清炒鸡毛菜 1份

晚点：
蛋挞 1个
牛奶 1杯

3.7 第7天饮食

早餐：
番茄鸡蛋面 1碗
苹果 1个

中餐：
米饭 1碗
香菇油菜 1份
莲子猪肚汤 适量

午点：
饼干 适量
牛奶 1杯

晚餐：
红豆黑米粥 1碗
木墀肉或南瓜炒肉丝 1份

晚点：
三丁豆腐羹 或 三丝黄花羹 适量

四、顺产分娩产后8至14天饮食

4.1 早餐

五更饭 或 核桃枸杞紫米粥 或 荔枝山药莲子粥 1碗
鸡蛋 1个
苹果 1个

4.2 午餐

米饭 或 五谷杂粮饭 或 黑糯米油饭 1碗
清炒瓜片 或 木耳土豆肉片 或 虾皮烧菜心 或 番茄炒蛋 1份
猪蹄茭白汤 或 酒酿豆腐汤 或 黄花菜卿鱼汤 适量
香蕉 1个

4.3 午点

花生红豆汤 或 糯米酒冲鸡蛋花 1碗

4.4 晚餐

红豆饭或千层饼适量 或 麻油面条 1碗
木楫肉或花椰菜炒蘑菇 或 鸭丝炒黄豆芽 1份
海带豆腐汤 或 卿鱼萝卜香菇汤 或 白萝卜蛭子汤 或 乳鸽滋养汤 适量

4.5 晚点

阳春面 1小碗
牛奶 1杯

五、顺产分娩产后15至21天饮食

5.1 早餐

杏仁提子粳米粥 或 红薯粥 或 紫米甜粥 1碗
鸡蛋1个 或 蒸蛋羹 1碗
苹果 1个

5.2 午餐

米饭 1碗
肉片炒青椒 或 土豆南瓜焖鸡肉 或 西兰花炒牛柳 1份
乌鸡栗子枸杞汤 或 姜枣枸杞乌鸡汤 或 麻油鸡或花生猪脚汤 适量

5.3 午点

花生红豆汤 或 藕汁 1碗
香蕉 1个

5.4 晚餐

烧饼1个红枣栗子粥 1碗
红烧茄子 或 豆角炒排骨 或 明虾烧豆腐 1份
桃仁莲藕汤 或 红枣乌鸡汤 或 猪脚姜 适量

5.5 晚点

木瓜西米露 或 生化汤 1碗

六、顺产分娩产后22至28天饮食

6.1 早餐

红豆燕麦红枣粥 或 莲子银耳粥 或 鸡蛋玉米羹 1碗
苹果 1个

6.2 午餐

米饭1碗
素什锦 或 炒三丝 或 鲜蔬炒群菇 或 黄芪蒸鲈鱼 或 菠菜炒猪肝 1份
胡萝卜牛蒡排骨汤 或 莲子薏米煲鸭汤 适量

6.3 午点

桂圆红枣汤 或 银杏雪梨汤 或 木瓜牛奶露 适量
香蕉 1个

6.4 晚餐

花卷1个 或 红豆饭 1碗
香菇油菜 或 上汤芦笋 1份
粟子黄鳝煲 或 黄山蒸鸽 适量
归芪鲈鱼汤 或 黑豆核桃煲瘦肉 或 黄芪老鸭汤 适量

6.5 晚点

木瓜西米露 或 黑芝麻花生粥 1碗

七、顺产分娩产后29至42天饮食

7.1 早餐

鸡肉三明治 1份
山药枸杞粥 或 什锦水果羹 1碗

7.2 午餐

五谷杂粮饭 1碗
番茄炒蛋 或 香煎黄花鱼 1份
苑菜锂鱼汤 适量

7.3 午点

莲子银耳汤1碗 或 竹荪红枣茶 1杯
猕猴桃 1个

7.4 晚餐

鲑鱼炒饭 1份
丝瓜炒虾仁 或 莲藕炖牛腩 适量
番茄玉米汤 或 虾皮萝卜丝汤 适量

7.5 晚点

木瓜牛奶露 或 荠菜魔芋汤 适量
